Portfólio SOC Analyst

Este repositório reúne cenários práticos de segurança baseados em atividades comuns de um Security Operations Center (SOC).

O objetivo deste projeto é demonstrar habilidades em monitoramento de segurança, investigação de incidentes, criação de regras de detecção e resposta a incidentes.

Habilidades demonstradas

Monitoramento de eventos de segurança

Investigação de incidentes

Análise de logs

Threat Hunting

Criação de regras de detecção em SIEM

Resposta a incidentes de segurança

Uso de frameworks como MITRE ATT&CK

Cenários documentados

Investigação de ransomware em servidor de arquivos

Análise de e-mails de phishing

Detecção de tentativa de brute force

Investigação de login suspeito

Threat hunting de execução suspeita de PowerShell

Playbooks de resposta a incidentes

Todos os cenários foram criados para fins educacionais, com dados simulados ou anonimizados.

Objetivo

Demonstrar conhecimento prático em operações de segurança (SOC), análise de eventos e resposta a incidentes.
